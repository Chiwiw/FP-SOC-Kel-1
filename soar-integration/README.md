# SOAR Integration

> [!NOTE]
> Modul/bagian ini dikerjakan oleh **Anggota 4**.

## Deskripsi
Folder ini digunakan untuk menyimpan kode program, skrip otomatisasi, playbooks, atau konfigurasi integrasi dengan platform SOAR (Security Orchestration, Automation, and Response) seperti Shuffle, Cortex XSOAR, dll.

## Rencana Integrasi
- Integrasi Wazuh Alerts ke SOAR
- Otomatisasi aksi respon (Blocking IP, Alerting Slack/Telegram, dll)
