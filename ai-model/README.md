# AI Model

> [!NOTE]
> Modul/bagian ini dikerjakan oleh **Anggota 3**.

## Deskripsi
Folder ini digunakan untuk menyimpan seluruh pekerjaan terkait pemodelan Machine Learning / Artificial Intelligence (AI) untuk Security Operations Center (SOC).

## Struktur Folder yang Disarankan
- `notebooks/` - Jupyter Notebook untuk eksplorasi dan training
- `src/` - Source code model dan pipeline data
- `models/` - Saved model file (e.g., `.pkl`, `.h5`, `.onnx`)
- `requirements.txt` - Dependensi library python yang digunakan
